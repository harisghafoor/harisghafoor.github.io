LSTM Encoded NBEATS Model for Electricity Demand Forecasting

2022
Deep Learning Time Series Forecasting

Development of a Network of Stacked Blocks of MLP in which each block is encoded by Recurrent Layer i.e. LSTM in order to accurately capture the different kinds of trends & seasonality in Time Series.

Project Highlights

  • Integrated LSTM encoders into the NBEATS architecture to better capture temporal dependencies
  • Improved forecasting accuracy compared to standard NBEATS implementations
  • Successfully applied to electricity demand prediction with varying seasonality patterns
  • Developed extensible framework for time series analysis that can incorporate various neural network architectures

Technologies Used

Python PyTorch Pandas NumPy Matplotlib
